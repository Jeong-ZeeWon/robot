# 시오니 v6

8살 아이도 쉽게 즐길 수 있는 무료 포켓 로봇 웹앱입니다. v6는 시오니가 사용자의 말과 돌봄 행동을 기억하고, 성격과 성장 레벨이 함께 바뀌도록 보강한 버전입니다.

실행 주소:

```txt
https://jeong-zeewon.github.io/robot/
```

## v6 핵심 기능

- 기억 카드 추가: 최근 대화, 돌봄, 감정, 휴식 기억을 최대 6개까지 저장
- 성장 XP와 v6 성장 레벨 추가
- 성격 스탯 추가: 다정함, 호기심, 용기, 반짝임
- 성격에 따른 현재 시오니 타입 표시
- 상태와 성격을 조합한 추천 인사이트 추가
- 기존 오늘의 별 조각 모험, 스티커, 배지, 방 꾸미기 유지

## v5에서 유지되는 핵심 기능

- 오늘의 별 조각 모험 추가
- 별 조각 보상 시스템 추가
- 스티커 도감 추가
- 배지 수집 기능 추가
- 시오니 방 꾸미기 기능 추가
- 효과음 켜기/끄기 버튼 추가
- 간단한 Web Audio API 효과음 추가
- 모험 완료 시 축하 이펙트 추가
- 기존 돌봄 버튼과 오늘의 모험을 자동 연동

## 오늘의 모험

오늘의 모험은 아래 행동을 완료하면 별 조각을 받을 수 있습니다.

- 시오니에게 안녕 인사하기
- 시오니 쓰담해주기
- 간식 주기
- 오늘 기분 고르기
- 시오니와 놀기

모험을 모두 완료하면 하루에 한 번 별 조각을 받을 수 있습니다.

## 보상 시스템

- 별 조각: 오늘의 모험 완료 보상
- 스티커: 특정 행동을 여러 번 하면 해금
- 배지: 별 조각을 모으면 해금
- 시오니 방: 별 조각 개수에 따라 배경이 바뀜

## 유지되는 기능

- 답변 다양화
- 최근 답변 반복 방지
- 반응 이펙트
- 움직임 애니메이션
- 배고픔, 에너지, 외로움, 친밀도, 기분 게이지
- 돌봄 행동 10초 쿨타임
- 목소리 설정
- 브라우저 localStorage 저장

## 무료 유지 방식

- 서버 없음
- 회원가입 없음
- API 키 없음
- OpenAI API 사용 없음
- 브라우저 localStorage와 Web Speech API, Web Audio API만 사용

## 주요 파일

```txt
index.html          v5 화면 구조
style.css           기본 디자인
motions.css         로봇 움직임 애니메이션
v4.css              한 화면 돌봄 UI
v5.css              오늘의 모험, 보물, 방 꾸미기 디자인
responses.js        답변 후보 은행
responses-extra.js  쓰다듬기 전용 답변 은행
v4.js               목소리 설정과 반응 이펙트
v5.js               별 조각 모험, 스티커, 배지, 효과음
v5-version.js       v6 표시 보정
app.js              반응 엔진, 게이지 밸런스, v6 기억/성장/성격
persist.js          상태 저장 보강
sw.js               오프라인 캐시
```

## 다운그레이드 방법

v6 작업은 `sioni-v6` 브랜치에서 진행할 수 있습니다. 마음에 들지 않으면 `main` 브랜치로 돌아가면 기존 v5 코드로 되돌릴 수 있습니다.
