# 10살 시오니

8살 아이도 쉽게 즐길 수 있는 무료 포켓 로봇 웹앱입니다. 이번 단계는 버전 번호보다 성장한 느낌을 우선해서, 화면에는 `10살 시오니`로 표현합니다.

실행 주소:

```txt
https://jeong-zeewon.github.io/robot/
```

## 핵심 변화

- 화면 표시를 버전명 대신 `10살 시오니` 중심으로 정리
- 터치 반응을 `아코야`, `간지러워`, `살살 만져줘`처럼 짧고 즉각적인 말투로 변경
- 간식 반응을 `냠냠`, `오물오물`, `우물우물` 같은 먹는 액션 말투로 변경
- 긴 설명형 문구보다 표정과 짧은 반응을 더 자주 사용
- 기존 40종 얼굴 표정과 먹는 표정 액션 유지
- UI 보정 CSS 추가로 카드, 버튼, 배경, 상태 영역을 더 부드럽게 정리
- 서버, API 키, 유료 AI 없이 브라우저 저장과 로컬 반응만 사용

## 유지되는 기능

- 대화/돌봄 기억
- 상태 기반 얼굴 표정
- 대기 중 자동 반응
- 배고픔, 에너지, 외로움, 친밀도, 기분 게이지
- 효과음과 기기 음성 설정
- 브라우저 localStorage 저장
- GitHub Pages 배포

## 주요 파일

```txt
index.html            화면 구조
style.css             기본 디자인
v5.css                로봇 방, 상태 카드, 기억 카드
sioni-10.css          10살 시오니 UI 보정
v9-face.css           40종 얼굴 표정 스타일
v9-face-engine.js     문맥 기반 얼굴 표정 엔진
responses.js          기본 대화 은행
responses-extra.js    기존 쓰다듬기 반응
sioni-10-lines.js     10살 시오니 짧은 반응 말투
app.js                상태, 게이지, 대화 흐름
idle.js               대기 중 상태 기반 반응
memory-engine.js      기억 요약
persist.js            상태 저장 보강
sw.js                 오프라인 캐시
```
