# 시오니 v2

터치와 말에 반응하는 무료 포켓 로봇 웹앱입니다.

실행 주소:

```txt
https://jeong-zeewon.github.io/robot/
```

## v2에서 추가된 기능

- 시간대별 인사
- 연속 방문일과 최고 연속 방문 기록
- 친밀도 레벨 표시
- 오늘의 미션 체크리스트
- 빠른 말걸기 버튼
- 더 많은 감정 반응: 기쁨, 슬픔, 졸림, 부끄러움, 놀람, 생각 중, 배고픔
- 오래 누르기와 빠른 연속 터치 반응
- 상태에 따라 바뀌는 배경 분위기
- v1 저장값을 v2로 자동 이전

## 무료 유지 방식

- 서버 없음
- 회원가입 없음
- API 키 없음
- OpenAI API 사용 없음
- 브라우저 localStorage만 사용
