# 시오니 v3

터치와 말에 반응하는 무료 포켓 로봇 웹앱입니다.

실행 주소:

```txt
https://jeong-zeewon.github.io/robot/
```

## v3에서 추가된 기능

- 답변 은행을 `responses.js`로 분리
- 카테고리별 답변 후보 10~15개 이상 추가
- 최근 사용 답변 반복 방지
- 같은 말에도 다른 표정, 다른 움직임, 다른 문장으로 반응
- 새 로봇 모션 파일 `motions.css` 추가
- 포만감 게이지 추가
- 외로움 게이지 추가
- 밥 주기, 놀아주기, 재우기에 쿨타임 적용
- 밥을 줘도 배고픔이 한 번에 해결되지 않도록 조정
- 재우기는 즉시 조금만 회복되고, 시간이 지나며 추가 회복
- 놀이가 기분은 올리지만 에너지와 배고픔을 함께 소모하도록 변경
- v1/v2 저장값을 v3로 자동 이전

## 무료 유지 방식

- 서버 없음
- 회원가입 없음
- API 키 없음
- OpenAI API 사용 없음
- 브라우저 localStorage만 사용

## 주요 파일

```txt
index.html      화면 구조
style.css       기본 디자인
motions.css     로봇 움직임 애니메이션
responses.js    답변 후보 은행
app.js          v3 반응 엔진과 게이지 밸런스
persist.js      상태 저장 보강
sw.js           오프라인 캐시
```
