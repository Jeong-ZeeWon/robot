# 시오니 v4.1

한 화면에서 터치와 선택으로 돌보는 무료 포켓 로봇 웹앱입니다.

실행 주소:

```txt
https://jeong-zeewon.github.io/robot/
```

## v4.1 변경 내용

- 탭으로 숨겨져 있던 영역을 한 화면에 펼침
- 돌봄 버튼 아래에 역할 설명 추가
- 사용자 화면에서 포만감 게이지 제거
- 상태 확인 문장에서 포만감 제거
- 목소리 기본값을 과한 고음에서 부드러운 톤으로 낮춤
- 목소리 모드 이름을 더 현실적으로 변경
- 음높이 조절 범위를 줄여 과한 고음 변조를 방지
- 기기와 브라우저에 들어 있는 음성 중에서 선택하는 방식으로 안내

## 돌봄 버튼 역할

- 쓰담: 외로움 감소, 친밀도 증가, 기분 증가
- 간식: 배고픔 감소, 쿨타임 있음
- 놀기: 기분 증가, 에너지 감소, 배고픔 증가
- 쉬기: 에너지 증가, 시간이 지나며 추가 회복

## 유지되는 기능

- 답변 다양화
- 최근 답변 반복 방지
- 반응 이펙트
- 움직임 애니메이션
- 배고픔, 에너지, 외로움, 친밀도, 기분 게이지
- 돌봄 행동 쿨타임
- 브라우저 저장 기능
