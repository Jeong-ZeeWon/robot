# 시오니 v4

채팅창 없이 터치와 선택으로 돌보는 무료 포켓 로봇 웹앱입니다.

실행 주소:

```txt
https://jeong-zeewon.github.io/robot/
```

## v4에서 추가된 기능

- 텍스트 입력창을 화면에서 제거하고 로봇 중심 UI로 변경
- 홈, 돌보기, 미션, 설정 하단 탭 UI 추가
- 감정 버튼으로 시오니에게 말 걸기
- 돌보기 버튼을 쓰담, 간식, 놀기, 쉬기로 재구성
- 시오니 추천 행동 카드 추가
- 목소리 설정 패널 추가
- 목소리 모드: 귀여운 시오니, 아주 작은 시오니, 차분한 시오니, 졸린 시오니, 신난 시오니
- 기기 음성 선택 기능 추가
- 음높이, 속도, 볼륨 조절 기능 추가
- 목소리 테스트 버튼 추가
- 감정에 따라 말투의 pitch, rate, volume이 자동 보정되도록 설정
- 반응 이펙트 추가: 하트, 별, Zzz, 물음표, 쿠키, 눈물방울
- 기존 v3의 답변 다양화, 반복 방지, 포만감/외로움, 쿨타임 기능 유지

## 무료 유지 방식

- 서버 없음
- 회원가입 없음
- API 키 없음
- OpenAI API 사용 없음
- 브라우저 localStorage와 Web Speech API만 사용

## 주요 파일

```txt
index.html          v4 화면 구조
style.css           기본 디자인
motions.css         로봇 움직임 애니메이션
v4.css              v4 전용 UI와 반응 이펙트
responses.js        답변 후보 은행
responses-extra.js  쓰다듬기 전용 답변 은행
v4.js               탭 UI, 목소리 설정, 반응 이펙트
app.js              반응 엔진과 게이지 밸런스
persist.js          상태 저장 보강
sw.js               오프라인 캐시
```
