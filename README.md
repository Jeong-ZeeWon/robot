# 시오니 v9

8살 아이도 쉽게 즐길 수 있는 무료 포켓 로봇 웹앱입니다. v9은 하나의 다정한 시오니 말투와 기억 엔진을 유지하면서, 얼굴 화면의 레트로 표정과 디테일을 크게 강화한 버전입니다.

실행 주소:

```txt
https://jeong-zeewon.github.io/robot/
```

## v9 핵심 기능

- 기억 엔진 추가: 감정 반복, 돌봄 패턴, 최근 신호를 따로 저장하고 요약
- 대화 엔진 확장: 키워드 반응에 현재 게이지와 기억 요약을 함께 반영
- 하나의 기본 말투 강화: 성격 타입을 늘리지 않고 다정한 시오니 톤을 유지
- v9 표정 엔진: 레트로 화면 스캔라인, 픽셀 글로우, 긴 표정 프레임 추가
- 30종 표정 변형: 같은 감정 안에서도 반복을 줄이고 다른 픽셀 장식이 나오도록 확장
- 슬픈 표정: 화면 안에서 픽셀 눈물이 흐르는 효과 추가
- 배고픈 표정: 화면 안에서 픽셀 침이 흐르는 효과 추가
- 졸림/기쁨/생각/놀람 표정의 입 모양과 눈 애니메이션 강화
- v8 액션 엔진 유지: 버튼 반응과 대기 모션에 고개/몸 동작 추가
- v7 기억 엔진 카드 유지: 대화 수, 기억 수, 돌봄 수, 대기 반응 수 표시
- 대기 중 상태 기반 반응 강화: 눈 깜빡임과 주변 보기 빈도 증가
- 별 조각 모험, 스티커, 배지, 방 꾸미기처럼 화면에서 의미가 흐린 보상 기능은 제거

## v5에서 유지되는 핵심 기능

- 효과음 켜기/끄기 버튼 추가
- 간단한 Web Audio API 효과음 추가
- 버튼 반응 애니메이션
- 돌봄 버튼과 감정 버튼

## 유지되는 기능

- 답변 다양화
- 최근 답변 반복 방지
- 반응 이펙트
- 움직임 애니메이션
- 배고픔, 에너지, 외로움, 친밀도, 기분 게이지
- 돌봄 행동 10초 쿨타임
- 목소리 설정
- 브라우저 localStorage 저장

## 무료 유지 방식

- 서버 없음
- 회원가입 없음
- API 키 없음
- OpenAI API 사용 없음
- 브라우저 localStorage와 Web Speech API, Web Audio API만 사용
